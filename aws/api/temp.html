Candidates - Job Applications

  • All job applications records for one candidate: /$context.stage/candidates/{accountName}/jobapplications
  • A specified job applications record for the specified candidate: /$context.stage/candidates/{accountName}/jobapplications/{id}
  • The most recenty inserted job applications record for the specified candidate: /$context.stage/candidates/{accountName}/jobapplications/inserted
Example URI's

GET

  • https://api.agility360app.net/beta/candidates/mcdaniel/jobapplications
  • https://api.agility360app.net/beta/candidates/mcdaniel/jobapplications/2
  • https://api.agility360app.net/beta/candidates/mcdaniel/jobapplications/inserted
